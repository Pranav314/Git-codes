command                                            ==                            task

git config --global user.name your name            ==  add the user name
git config --global user.email youremail@gmail.com ==  add email of the user
git config --global user.name                      ==  shows the name of current user
git config --global user.email                     ==  shows the email of the current user
git init                                           ==  initialise the repository
git status                                         ==  shows the status of the files in the repository
git add NameOfTheFile.format                       ==  Tracks a file
git add -A                                         ==  Tracks all the file present in the repository
touch NameOfTheFile.format                         ==  creates a new empty file in the repository
git commit -m "Messsage"                           ==  commits the repository
git checkout NameOfTheFile.format                  ==  restores the previous data in the file
git checkout -f                                    ==  restores the previous data present in all the files of the repository
git log                                            ==  shows the date when you had made changes to the files and name of the author,email and the message of change
git log -p -1(number of files)                     ==  shows the changes
